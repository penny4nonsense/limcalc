# Revision history for limcalc

## 0.1.0.0 -- 2026-07-07

* First release.
* Log-Puiseux series representation for local expansion of elementary
  and special functions.
* Certified limit computation with result types Exists, Pole,
  DoesNotExist, and LimitError.
* Symbolic differentiation via coefficient extraction; gradient,
  Jacobian, and Hessian without additional machinery.
* Risch algorithm integration for rational, exponential, and
  logarithmic cases; Euler substitution for trigonometric integrands.
* NonElementary as a proved certificate of non-existence.
* Special function recognition: erf, Si, Ci, Ei, li.
* JSON line protocol CLI (limcalc-cli).
